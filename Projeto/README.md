# ✝️ Portal Católico Integrado

## 📖 Sobre o Projeto

O **Portal Católico Integrado** foi desenvolvido como atividade do **Módulo 3 – Low Code / No Code / Vibecode**, com o objetivo de criar uma plataforma centralizada para auxiliar a comunidade cristã no acesso a conteúdos religiosos e na organização de atividades litúrgicas.

A aplicação reúne funcionalidades para orações, repertório musical e gerenciamento de missas em um único ambiente digital, proporcionando praticidade e melhor experiência para os usuários.

---

## 🎯 Objetivo

Desenvolver uma plataforma web capaz de centralizar conteúdos e recursos utilizados pela comunidade católica, facilitando o acesso a:

* Orações;
* Salmos;
* Músicas litúrgicas;
* Informações sobre missas.

---

## 🖥️ Funcionalidades

### 📿 Rezas

Área dedicada à consulta de orações católicas.

### 🎵 Salmos

Seção para visualização e organização de salmos utilizados nas celebrações.

### 🎶 Músicas

Catálogo de músicas litúrgicas com sistema de pesquisa dinâmica.

### ⛪ Missas

Gerenciamento de horários e informações relacionadas às celebrações.

---

## 🔍 Destaque do Projeto

Foi implementado um sistema de busca dinâmica na seção de músicas, permitindo localizar canções por:

* Título;
* Artista.

Essa funcionalidade facilita a montagem de repertórios e a localização rápida de músicas para diferentes momentos litúrgicos.

---

## ⚙️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando:

* HTML5
* CSS3
* JavaScript
* Inteligência Artificial Generativa (apoio ao desenvolvimento)

---

## 🎨 Justificativa da Escolha Tecnológica

A utilização dessas tecnologias permitiu:

* Maior controle sobre a identidade visual do portal;
* Aplicação da temática visual em tons de verde e dourado;
* Navegação rápida e fluida;
* Execução leve diretamente no navegador;
* Implementação de funcionalidades interativas por meio do JavaScript.

Além disso, o modelo **Single Page Application (SPA)** proporciona uma experiência mais dinâmica ao usuário, eliminando recarregamentos desnecessários da página.

---

## 🚀 Vantagens Identificadas

### ⚡ Velocidade de Desenvolvimento

O uso de templates e ferramentas de IA Generativa acelerou significativamente a criação da interface e das funcionalidades.

### 🎨 Interface Personalizada

A personalização visual permitiu adaptar o portal à identidade litúrgica desejada.

### 📚 Centralização de Conteúdo

Diversos recursos religiosos foram reunidos em uma única plataforma, facilitando o acesso dos usuários.

---

## ⚠️ Limitações Encontradas

### 💾 Armazenamento Temporário

Os dados relacionados ao agendamento de missas não são persistidos após a atualização da página, devido à ausência de um banco de dados.

### 🎵 Escalabilidade do Acervo Musical

O gerenciamento manual das músicas diretamente no código pode se tornar complexo conforme o acervo cresce.

### 🔗 Interatividade Limitada

Atualmente não há integração com APIs externas para obtenção automática de letras, cifras ou conteúdos musicais.

---
